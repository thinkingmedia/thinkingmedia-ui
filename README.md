## Building

This library requires NodeJS, NPM, Grunt, Bower and Compass to successfully build.

```
$ git clone https://github.com/thinkingmedia/thinkingmedia-ui thinkingmedia-ui
$ cd thinkingmedia-ui
$ npm install
$ bower install
$ grunt build
```
