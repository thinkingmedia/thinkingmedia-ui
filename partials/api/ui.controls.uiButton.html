uiButton
service in module ui.controls

Description

Handles the rendering of a UI button.

Size Inheritance

Buttons, Toolbars and other UI controls can be shown using different sizes. When a uiButton is a child of uiToolbar it can inherit the size setting from uiToolbar.

Dependencies

  • uiSize

Usage

as element:
<ui-button
       size="{string}">
</ui-button>

Parameters

ParamTypeDetails
sizestring

The size for the button (can be inherited)
