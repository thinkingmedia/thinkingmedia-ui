uiDelayHover
service in module ui.components

Description

Executes the expression after the mouse hovers over the element for 500ms or the value of ui-delay-time attribute.

Usage

as attribute
<ANY ui-delay-hover="{expression}"
     [ui-delay-time="{string}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
uiDelayHoverexpression

The expression to execute after hover timeout.

uiDelayTime
(optional)
string

How long to delay before executing the expression (default is 500ms).
